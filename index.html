AC自动机

AC自动机

　　AC自动机是用来求一个串中出现了多少个模式串的算法。

###fail数组
　　类比KMP中的next数组，即在Trie上，i点向上与fail[i]到根节点之前组成的字符串长度相同组成的字符串相等。

###算法实现
　　步骤为：按照模式串建立一个Trie树，Trie上的每一个结点有一个fail值，将所求串在Trie上比对。
　　
　　建立Trie树
　　在每个模式串的结尾的结点上记录一个count，并把它 + 1。

　　如何求fail？
　　root的fail为0，从根节点开始Bfs，检查当前点(设为i)的每一个子节点(设为代表的字母为j)，检查fail[i]是否存在一个为j的子节点，有则为fail[i]的j子节点，没有则为根节点。

　　比对
　　将所求串在Trie上跑，再从当前位置跑一次fail数组，并把答案加上count，下次不再跑加过的count。

扩展KMP

来源: http://www.cppblog.com/MatoNo1/archive/2011/04/17/144390.aspx

拓展KMP

　　拓展KMP是用来求两个串(设为A串B串)中，A串的每一位开始，最多能与B串从前缀开始匹配多少位。

　　
设ex[i]为A从第i位开始最多能匹配ex[i]位的B前缀，next[i]为B从第i位开始最多能匹配next[i]位的B的前缀。
　　
假设现在匹配到第i位，在k时匹配到最远(即k + ex[k] - 1最大时),设p为这个最远的距离(即p = k + ex[k] - 1)。
　　
A[k..p] = B[1..p - k + 1]
$\Rightarrow$ A[i..p] = B[i - k + 1..p - k + 1]

按next[i - k + 1]分类讨论(设next[i - k + 1] = L)
1° L <= p - k + 1时
B[1..L] = B[i - k + 1..i - k + L] = A[i..i + L - 1]
因为next的定义，A不可能再匹配更多的B，即ex[i]=L

2° L > p - k + 1时
A[i..p] = B[1..p - i + 1]
即可以从A[p]开始与B[p - i + 1]匹配。

注意：p有可能小于i，根据p的定义，此时k = i - 1, p = i - 2，则从i开始匹配即可。

树链剖分

树链剖分

点分治

E.g. JZOJ 1166
　　给出一棵带边权的树，问有多少对点的距离<=Len。

　　一棵树的重心表示一个以它为根，每个子树的大小不超过size / 2的点。
　　怎么找重心？将其遍历两次，一次计出每个子树的大小，第二次找出每个点的子树大小与除了这棵子树的点数中的较小值小于size/2的即为重心。
　　每次找树的重心，然后暴力出每个点到重心的距离，排序，O(n)扫出两个点间的距离 <= len 的点的对数，再对重心的每个子树做一次相同的操作。
